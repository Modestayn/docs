Урок 5 – Цикли, масиви та спред-оператор
У цьому уроці розглянемо цикли, базові операції з масивами, поняття мутації коду та використання спред-оператора.

1. Цикли
for
Виконує код певну кількість разів, поки умова істинна.

for (let i = 0; i  5; i++) { console.log(i); }
while
Виконує код доти, доки умова істинна.

let i = 0; while (i  5) { console.log(i); i++; }
do...while
Виконує код мінімум один раз, а потім перевіряє умову.

let i = 0; do { console.log(i); i++; } while (i  5);
for...of
Зручний для перебору елементів масиву.

const arr = [1,2,3]; for (const num of arr) { console.log(num); }
for...in
Використовується для перебору ключів об'єкта.

const obj = {a1,b2}; for (const key in obj) { console.log(key, obj[key]); }
2. Масиви (базове)
Масив – структура для зберігання колекції даних.

const fruits = ['apple','banana','orange']; console.log(fruits[0]);
3. Спред-оператор (...)
Дозволяє розгортати масив чи об'єкт.

const arr1 = [1,2]; const arr2 = [...arr1,3,4]; console.log(arr2);  [1,2,3,4]
4. Мутація
Мутація означає зміну існуючого об'єкта чи масиву напряму.

const arr = [1,2]; arr.push(3); console.log(arr);  [1,2,3] – масив мутував
Щоб уникати мутацій, використовують копії

const arr = [1,2]; const newArr = [...arr,3]; console.log(newArr);  [1,2,3]
Підсумок
Цикли дозволяють багаторазово виконувати код.
Масиви зберігають послідовність даних.
Спред-оператор розгортає структури.
Мутація змінює існуючі дані напряму, що не завжди добре.
