Завдання:

Створити базовий HTML/CSS макет сайту

Хедер із назвою сайту.

Основний контент із текстом і кількома блоками.

Футер із автором або копірайтом.

Додати кнопку перемикання теми

Кнопка повинна знаходитися у хедері.

При натисканні змінює стиль сайту між світлою та темною темою.

Реалізація перемикання теми через JS

Використовувати клас на <body> (наприклад, dark-theme) для темної теми.

Для світлої теми цей клас прибирається.

Змінювати фонові кольори, кольори тексту, посилань тощо через CSS.

Зберігання вибору користувача

Використати localStorage, щоб при перезавантаженні сторінки тема залишалася обрана користувачем.

Додатково (опційно для кращих студентів)

Додати плавний перехід між темами через CSS (transition).

Додати іконки сонця/місяця на кнопці для візуального ефекту.

Зробити, щоб тема змінювалася автоматично за часом доби (день/ніч).
