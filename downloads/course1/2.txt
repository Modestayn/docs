Lesson 2 – Основи CSS та методологія БЕМ



У цьому уроці ми познайомились із CSS (Cascading Style Sheets) – мовою стилів,
яка відповідає за зовнішній вигляд HTML-елементів.





1. Підключення CSS

<head>
  <link rel="stylesheet" href="style.css"> <!-- Зовнішній CSS (коментар замість посилання) -->
</head>

<style> /* Вбудований CSS */
p { color: blue; }
</style>

<p style="color: red;">Інлайн-стиль</p> <!-- Найменш бажаний -->




2. Методологія БЕМ (Block – Element – Modifier)


БЕМ допомагає організувати класи в HTML і CSS, щоб код був зрозумілий і масштабований.


<!-- Блок (незалежна сутність) -->
<div class="card">

<!-- Елемент (частина блоку) -->
<h2 class="card__title">Назва</h2>
<p class="card__text">Опис</p>

<!-- Модифікатор (змінює вигляд блоку) -->
<button class="card__btn card__btn--primary">OK</button>

</div>




3. Основний тег для стилізації – <div>


Тег <div> – контейнер без власного значення.
Використовується для групування елементів і створення структур сторінки.





4. Кольори

/* CSS-формати кольорів */
color: red;             /* назва кольору */
color: #ff0000;         /* HEX */
color: rgb(255, 0, 0);  /* RGB */
color: rgba(255,0,0,0.5); /* з прозорістю */
color: hsl(0, 100%, 50%); /* HSL */




5. Одиниці вимірювання


px – пікселі (фіксована величина).

% – відсотки від батьківського елемента.

em – відносно шрифту батьківського елемента.

rem – відносно шрифту кореня (<html>).

vh – відсоток від висоти вікна браузера.

vw – відсоток від ширини вікна браузера.



.box {
  width: 200px;   /* пікселі */
  height: 50%;    /* половина від контейнера */
  font-size: 2em; /* у 2 рази більше за батьківський шрифт */
  margin-top: 5vh; /* 5% від висоти екрана */
}




6. Основні властивості

.block {
  width: 300px;       /* ширина */
  height: 150px;      /* висота */

  background: lightblue; /* фон */
  border: 2px solid black; /* рамка */

  margin: 20px;       /* зовнішні відступи */
  padding: 15px;      /* внутрішні відступи */
}



width / height – розміри елемента.

background – фон (колір, градієнт, картинка).

border – рамка (товщина, стиль, колір).

margin – відступ ЗОВНІ елемента.

padding – відступ ВСЕРЕДИНІ елемента.





7. Приклад стилізації

<div class="card card--highlight">
  <h2 class="card__title">Заголовок</h2>
  <p class="card__text">Це приклад картки з CSS-оформленням.</p>
</div>

/* CSS */
.card {
  width: 250px;
  padding: 20px;
  background: #f0f0f0;
  border: 1px solid #ccc;
  margin: 10px auto;
}

.card--highlight {
  background: lightyellow;
  border: 2px dashed orange;
}

.card__title {
  font-size: 1.5rem;
  margin-bottom: 10px;
}

.card__text {
  font-size: 1rem;
}
