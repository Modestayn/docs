Урок 15 – Фінальний тест: JavaScript, React, TypeScript та суміжні технології
Цей тест перевіряє знання студентів у всіх ключових темах курсу: JS, DOM, HTML/CSS, Flexbox, Git, React, TypeScript, TailwindCSS, Redux Toolkit, Axios, i18n та React Router. Студенти повинні дати теоретичні відповіді та виконати практичні завдання.

JavaScript та DOM
Що таке DOM і як з ним взаємодіяти у JavaScript?
Напишіть код для додавання нового елемента <li> до списку <ul id='list'>.
Чим відрізняються let, const та var?
Опишіть різницю між масивом та об’єктом у JS. Наведіть приклади.
Практичне завдання: Напишіть функцію, яка приймає масив чисел і повертає відсортований масив.
HTML/CSS та Flexbox
Що таке Flexbox? Напишіть приклад верстки з його використанням.
Які є основні властивості Flexbox-контейнера та їх призначення?
Створіть HTML-форму з полями: ім’я, email та кнопкою “Відправити”.
Практичне завдання: Верстати просту сторінку з навігаційним меню за допомогою Flexbox.
Git та GitHub
Поясніть основні команди Git: git add, git commit, git push.
Що таке git merge та git rebase?
Практичне завдання: Ініціалізуйте Git-репозиторій, додайте файл index.html і зробіть коміт.
React та суміжні технології
Напишіть приклад простого React-компонента, що відображає “Привіт, світ!”.
Що таке props і state в React? Поясніть відмінність.
Як працює useEffect? Наведіть приклад використання.
Практичне завдання: Створіть React-компонент To-Do List з можливістю додавати та видаляти завдання.
TypeScript
Чим TypeScript відрізняється від JavaScript?
Оголосіть тип для масиву рядків у TypeScript.
Практичне завдання: Напишіть інтерфейс User з полями id, name та email.
TailwindCSS
Які переваги TailwindCSS перед звичайним CSS?
Напишіть приклад кнопки, стилізованої за допомогою TailwindCSS.
Практичне завдання: Створіть картку профілю користувача з TailwindCSS.
Redux Toolkit (RTK)
Що таке Redux Toolkit і чому його використовують?
Напишіть приклад створення slice в RTK.
Практичне завдання: Реалізуйте лічильник за допомогою Redux Toolkit.
Axios та робота з API
Як за допомогою Axios виконати GET-запит? Напишіть приклад.
Що таке interceptors в Axios?
Практичне завдання: Напишіть функцію, що виконує POST-запит через Axios.
i18n, React Router
Що таке i18n і як його застосувати в React?
Напишіть приклад налаштування маршруту в React Router.
Практичне завдання: Реалізуйте навігацію між двома сторінками за допомогою React Router.
