  • Питання - відповідь 2025-09-22

    HTML + CSS Code:

    
<!DOCTYPE html>
<html lang="uk">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <style>
    .box {
      width: 300px;
      height: 200px;
      background: lightblue;
    }

    /* Mobile */
    @media (max-width: 480px) {
      .box {
        width: 100%;
        background: lightcoral;
      }
    }
  </style>
</head>
<body>
  <div class="box">Адаптивний блок</div>
</body>
</html>

    HTML + CSS. Який колір фону буде у блоку, якщо відкрити сторінку на екрані шириною 400px?




    Що робить код:

    - Є блок з класом .box, за замовчуванням він має ширину 300px і фон lightblue. - Додається медіа-запит @media (max-width: 480px), який змінює стилі для екранів до 480px шириною. - У цьому випадку блок отримує width: 100% і фон lightcoral.

    Чому саме такий результат:
    - При ширині екрану 400px умова (max-width: 480px) виконується. - Тому застосовуються стилі з медіа-запиту. - Отже, фон блоку буде lightcoral.

  • Адаптивність сайту: meta-теги та media queries 2025-09-22

    Сучасні сайти мають виглядати зручно як на телефоні, так і на великому моніторі. Для цього використовується адаптивна верстка. Перший крок — додати у <head> спеціальний meta-тег: 

    <meta name="viewport" content="width=device-width, initial-scale=1.0">

    Цей тег каже браузеру підлаштовувати ширину сайту під ширину пристрою. Без нього дизайн може виглядати зламано на мобільних.

    Далі використовуються CSS media queries, які дозволяють змінювати стилі залежно від ширини екрана. Наприклад, можна задати різні відступи чи розміри шрифту для мобільних, планшетів та десктопів.

    Найпопулярніші брейкпоінти (з прикладу на зображенні): 

    /* Mobile */
@media (max-width: 480px) {
  /* styles */
}

/* Extra small devices */
@media (min-width: 481px) and (max-width: 767px) {
  /* styles */
}

/* Small tablets */
@media (min-width: 768px) and (max-width: 991px) {
  /* styles */
}

/* Large tablets/laptops */
@media (min-width: 992px) and (max-width: 1199px) {
  /* styles */
}

/* Desktops */
@media (min-width: 1200px) and (max-width: 1919px) {
  /* styles */
}

/* Extra large screens */
@media (min-width: 1920px) {
  /* styles */
}

    Завдяки такій системі можна контролювати вигляд сайту під будь-який пристрій.

    Порада: починайте верстати з «mobile-first» підходу — спершу для телефонів, а потім додавайте стилі для більших екранів.

  • Як працює Google Fonts та семантика в HTML 2025-09-22

    Майже кожен сайт використовує кастомні шрифти, і найзручніший сервіс для цього — Google Fonts. Він безкоштовний, швидкий та простий у використанні.

    Щоб підключити шрифт, зайдіть на Google Fonts, оберіть потрібний стиль (наприклад, Roboto чи Montserrat), згенеруйте лінк і вставте його в <head> вашої HTML-сторінки: 

    <link href="https://fonts.googleapis.com/css2?family=Roboto&display=swap" rel="stylesheet">

    Далі у CSS просто використовуйте: 

    body {
    font-family: 'Roboto', sans-serif;
}

    Окрім шрифтів, важливу роль у вебі відіграє семантика. Це правильне використання HTML-тегів, щоб код був зрозумілий не лише людям, а й пошуковикам та скрінрідерам.

    Наприклад:

    • <header> — для шапки сайту
    • <main> — основний контент
    • <article> — окрема стаття
    • <section> — розділ сторінки
    • <footer> — підвал сайту

    Порада: поєднуйте Google Fonts із семантичними тегами — ваш сайт виглядатиме сучасно та буде SEO-дружнім.

  • Питання - відповідь 2025-09-20

    JavaScript Code:

    
let users = [
    { id: 11, name: 'Adam', age: 23, group: 'editor' },
    { id: 47, name: 'John', age: 28, group: 'admin' },
    { id: 85, name: 'William', age: 34, group: 'editor' },
    { id: 97, name: 'Oliver', age: 28, group: 'admin' }
];

let res = users.filter(it => it.group === 'admin');

console.log(res);

    JavaScript. що виведе в консоль?




    Що робить код users — масив об’єктів, кожен об’єкт містить id, name, age та group. filter створює новий масив, який включає ті об’єкти, у яких group === 'admin'. console.log(res) виводить цей новий масив у консоль. Чому саме такий результат У початковому масиві є 4 користувачі, але лише 2 з них мають group: 'admin' — це John та Oliver. Метод filter повертає новий масив, не змінюючи початковий. Тому в консоль потрапляє масив із цих двох об’єктів.

  • Тайм-менеджмент для розробників-початківців 2025-09-10

    Якщо ви вчите Frontend чи будь-яку іншу IT-спеціальність, швидко помічаєте: завдань багато, а часу завжди мало. Ключ до успіху — правильний тайм-менеджмент.

    Використовуйте метод Pomodoro (25 хвилин роботи + 5 хвилин відпочинку), щоб уникати перевтоми. Плануйте тиждень у Notion або Trello — створюйте борди з темами, які потрібно вивчити (HTML, CSS, JS, Git).

    Не забувайте про читання офіційної документації. Наприклад, MDN для JavaScript чи React Docs — це джерела, яким довіряють навіть сеньйори.

    Порада: починайте день із завдання, яке найбільше «болить». Закриєте його — решта піде легше.
